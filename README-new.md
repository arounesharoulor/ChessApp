# Chess Tournament Manager

A lightweight Svelte app for managing chess players, tournaments, random pairings, and rankings.

## Features

- Create, edit, and delete players
- Create, edit, and delete tournaments
- Add or remove players from tournaments
- Generate random matches and record winners
- View final rankings for 1st, 2nd, and 3rd place
- Persist data in the browser with local storage

## Run locally

```bash
npm install
npm run dev
```

Open the local Vite URL shown in the terminal to use the app.
